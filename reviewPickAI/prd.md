[PRD] 리뷰픽 AI (ReviewPick AI): 마케터용 리뷰 분석 솔루션

1. 프로젝트 개요
이름: 리뷰픽 AI (ReviewPick AI)

한 줄 설명: 쿠팡·네이버 쇼핑 리뷰를 수집하여 AI가 긍/부정 수치화, 키워드 마인드맵, 마케팅 인사이트를 도출하는 전문가용 분석 도구.

유형: 크롬 확장 프로그램 (Chrome Extension - Manifest V3)

난이도: 상 (Advanced)

1. 사용자 시나리오
누가: 이커머스 상품 기획자(PM), 퍼포먼스 마케터, 브랜드 매니저.

언제: 경쟁사 제품의 약점을 파악하거나, 신규 광고 카피를 위한 페르소나의 '찐 언어'가 필요할 때, 혹은 주간 VOC 리포트를 작성할 때.

왜: 수천 개의 리뷰를 직접 읽는 비효율을 제거하고, AI를 통해 정량적(긍부정 %)과 정성적(마인드맵) 데이터를 즉시 얻기 위해.

1. 핵심 기능 목록
[필수 기능]
스마트 크롤러: 쿠팡(상품평), 네이버(쇼핑) 리뷰 데이터 자동 수집.

AI 인사이트 요약: Gemini 3 Flash Preview를 통한 핵심 3줄 요약 및 소구점 제안.

감성 분석: 전체 리뷰의 긍정/부정 비율(%) 시각화.

키워드 마인드맵: 빈출 키워드 간의 연관 관계 시각화.

[선택 기능 - 모두 포함]
멀티 포맷 내보내기: 분석 리포트(PDF), 로우 데이터(CSV), 마인드맵(PNG) 저장.

벤치마킹 비교 모드: 두 개 이상의 상품 URL 리뷰 동시 분석 및 비교.

스마트 키워드 필터: 특정 키워드 포함 리뷰만 필터링하여 AI가 재분석 및 수치 재계산.

1. 기술 스택
Extension: React, Tailwind CSS, Manifest V3.

AI Engine: Google Gemini 3 Flash Preview API.

Visualization: D3.js (마인드맵), Chart.js (감성 차트).

Language: TypeScript.

Storage: Chrome Storage API (임시 데이터 보관).

1. 화면 구성
팝업 창: 현재 페이지 수집 가능 여부 확인 및 [분석 시작] 버튼.

메인 대시보드 (Full Tab):

헤더: 상품 정보 및 데이터 추출(Export) 버튼.

요약 섹션: 긍/부정 도넛 차트 및 AI 요약 카드.

중앙 섹션: 인터랙티브 마인드맵 캔버스.

사이드바: 키워드 필터 입력 및 비교 모드 관리.

리스트: 감성 태깅이 완료된 원본 리뷰 목록.

1. 상세 기능 명세
동적 수집 로직: 쿠팡의 무한 스크롤 및 네이버의 페이지네이션을 자동으로 처리하여 텍스트를 파싱함.

Gemini 프롬프트 엔지니어링: 수집된 리뷰 텍스트를 Gemini 3 Flash에 전달하여 구조화된 JSON 형태로 요약, 키워드, 감성 수치를 반환받음.

마인드맵 인터랙션: 각 노드 클릭 시 연관된 하위 키워드를 펼치고, 클릭한 키워드가 포함된 리뷰 원문을 하이라이트함.

비교 엔진: 두 상품의 긍정 포인트를 대조하여 차별화 요소를 자동 도출.

1. 디자인 가이드
컨셉: Toss-Style (Minimal & Trust).

컬러: Primary: #0051FF (Toss Blue), Negative: #F04452, Background: #F2F4F6.

UI 요소: 넉넉한 여백, 16px 이상의 라운드 코너, 직관적인 아이콘(Lucide-react).

시각화: 복잡한 선 대신 부드러운 곡선을 사용한 마인드맵 레이아웃.

1. 제약 사항 (업데이트)
스크래핑 방어 기제: 쿠팡/네이버의 봇 탐지 정책으로 인해 너무 빠른 수집은 IP 차단을 유발함. (사람의 동작을 흉내 내는 Random Delay 필수).

데이터 무결성(할루시네이션): AI가 가상의 리뷰를 만들어내는 것을 방지하기 위해, 반드시 수집된 텍스트 범위 내에서만 요약하도록 시스템 프롬프트 제약.

브라우저 리소스: 수천 개의 리뷰를 한 페이지에 로드할 경우 메모리 과부하 발생. (데이터 처리 시 100개 단위의 청크 처리가 권장됨).

API Rate Limit: Gemini 3 Flash Preview의 분당 요청 제한을 고려하여, 대량의 리뷰는 컨텍스트 윈도우를 최대한 활용해 단일 요청으로 처리.

보안 및 저작권: 수집된 리뷰 데이터를 외부 서버에 무단 저장하거나 상업적으로 재판매하는 것에 대한 법적 책임 고지(Disclaimer) 필요.
