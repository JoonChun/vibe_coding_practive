## 2. 영역별 명세

### 2.1 헤더 (단청 5색 — 변경 없음)

현행 `h-12` (48px) 그대로 유지. 오방색 5등분 + 중앙 타이틀 + 우측 SSE 연결 indicator. page.tsx DANCHEONG_COLORS 배열 변경 없음.

### 2.2 두루마리 (TaskScroll — 변경 없음)

현행 `h-16` (64px), 한지 베이지 + 명조체 + 진행도 블록. 변경 없음.

### 2.3 어전 영역 (메인) ★ Palace Scene

어전 영역은 `flex-1 relative overflow-hidden`. 전체 화면에서 헤더·두루마리·KingInput을 뺀 나머지 전부를 차지함. 1280×800 기준 약 632px 높이 → 전체의 79%.

#### 좌·우 두꺼운 단청 기둥

IlwolObongdo.tsx SVG 안에서 처리. 좌 기둥 `x=0~60`, 우 기둥 `x=940~1000`. 화면 폭 1000단위 기준 각 6%.

- 기둥 몸체: 적색 `#C0392B`
- 기둥 내 단청 띠: 상단부에 청람·자황·흑 3단 가로 띠 (아래 SVG 명세 참조)
- 기둥 바닥 받침: 갈색 `#5C3A21`

#### 상단 대들보 + 단청 가로 띠

어전 영역의 최상단 영역. SVG y=0~50 (viewBox 기준).

- 오방색 5등분 가로 띠: 각 200단위 폭, 높이 50단위
- 대들보 하단 마감선: y=50, height=3, fill=`#1A1410` (진흑)
- 이 대들보는 헤더 오방색과 *별개*. 헤더는 HTML, 대들보는 SVG 안.

#### 후벽: 일월오봉도 전체 배경

기존 `x=360~640` 작은 병풍 액자 → **전면 삭제**하고 어전 내부 공간 전체(`x=60~940, y=53~450`)를 일월오봉도로 채움.

- 하늘 그라디언트: 상단 `#1A2B47` → 중단 `#3B5E8C`
- 달 (좌 상단): `cx=200 cy=150 r=48` — 기존 r=20에서 2.4배 확대
- 해 (우 상단): `cx=800 cy=150 r=48` — 기존 r=18에서 2.7배 확대
- 5봉 실루엣: 어전 내부 폭 880단위에 걸쳐 넓게 배치, 중앙 봉우리 높이 160px (기존 90px의 1.8배)

#### 매니저·왕 캐릭터 좌표 (어전 영역 내부 상대%)

```
캐릭터        left    top     비고
────────────────────────────────────────────────────
왕            50%     70%     마룻바닥 시작 직전. transform: translate(-50%, -100%)
정도전(좌외)  18%     75%     품계석 포함. transform: translate(-50%, -100%)
정약용(좌내)  32%     72%     품계석 포함. transform: translate(-50%, -100%)
장영실(우내)  68%     72%     품계석 포함. transform: translate(-50%, -100%)
이순신(우외)  82%     75%     품계석 포함. transform: translate(-50%, -100%)
```

> top 값이 어전 영역의 ~70~75% 라인 = 마룻바닥 시작선 바로 위.  
> transform `translate(-50%, -100%)` → 발 끝이 top 좌표에 닿는 구도.

#### 품계석

각 매니저 캐릭터 발 바로 아래 작은 직사각 돌.

```
크기:       24×16px (픽셀 SVG 또는 div)
색상:       fill=#6B7280 (회색 돌)
테두리:     stroke=#4B5563 strokeWidth=1
위치:       캐릭터 컴포넌트 내부 absolute, bottom=-16px, left=50%, transform=translateX(-50%)
왕 품계석:  왕 전용 옥좌 플랫폼 (120×24px)으로 대체 — 품계석보다 큰 단상
```

#### 마룻바닥

어전 영역 하단 25%. SVG y=450~600 (viewBox 600 기준).

- 바닥 기본색: `#5C3A21`
- 마룻결 가로선 4개: y=475, 510, 545, 575 / stroke=`#3D2615` strokeWidth=2
- 마루 상단 경계선: y=449, height=2, fill=`#1A1410`

#### 도제 등장 영역

```
등장 방향:
  좌측 도제 (정도전·정약용 휘하) → 화면 왼쪽 바깥 x=-60px 에서 슬라이드 인
  우측 도제 (장영실·이순신 휘하) → 화면 오른쪽 바깥 x=화면폭+60px 에서 슬라이드 인

착지 위치:
  해당 매니저 좌표 기준 발 오프셋 ±40px 클러스터링 (현행 getDojeOffsets 로직 유지)

도제가 기둥 뒤에 가리지 않도록:
  기둥 z-index: 5 / 도제 z-index: 8 / 매니저 z-index: 10 / 왕 z-index: 20
  → 도제가 기둥 뒤에서 나오는 연출 가능 (기둥이 도제를 살짝 가림)
```

### 2.4 KingInput

어전 영역 *바깥* 하단 고정. `shrink-0 h-14` (56px).

임금 분부 "어전 70%+ 차지"는 어전 영역 자체 비율 기준 → KingInput은 어전 영역 밖이므로 충돌 없음. 현행 disabled placeholder 유지. 위치·크기 변경 없음.

## 4. 매니저·왕 좌표 재배치

### 마룻바닥 비율과 캐릭터 발 라인

```
SVG viewBox 600 기준:
  마룻바닥 시작: y=450 / 전체 600 = 75%
  어전 영역 내 상대 top 75% = 마룻바닥 시작선

캐릭터 발 라인 목표: 마룻바닥 *위에 서있는* 모양
  → top: 75% (정도전·이순신), top: 72% (정약용·장영실), top: 70% (왕)
  → transform: translate(-50%, -100%) 적용 시 캐릭터 하단이 해당 라인에 닿음
```

### 캐릭터 절대 좌표 (page.tsx MANAGER_LAYOUT 갱신 대상)

```javascript
// 현행 v1 좌표 → v2 갱신 좌표
// ─────────────────────────────────────────────────
// 왕 (KingCharacter 위치 div)
//   v1: left:"50%", top:"35%"
//   v2: left:"50%", top:"70%"    transform: translate(-50%,-100%)

// 정도전
//   v1: left:"18%", top:"40%"
//   v2: left:"18%", top:"75%"    transform: translate(-50%,-100%)

// 정약용
//   v1: left:"30%", top:"45%"
//   v2: left:"32%", top:"72%"    transform: translate(-50%,-100%)

// 장영실
//   v1: left:"70%", top:"45%"
//   v2: left:"68%", top:"72%"    transform: translate(-50%,-100%)

// 이순신
//   v1: left:"82%", top:"40%"
//   v2: left:"82%", top:"75%"    transform: translate(-50%,-100%)
```

### 품계석 배치 다이어그램

```
어전 영역 내 y=75% 라인 (마룻바닥 시작) 전후:

  y=70%  [왕 캐릭터 하단]
         ┌──────────────────┐  ← 옥좌 플랫폼 (120×24px, #8B5C1E)
         └──────────────────┘
  y=75%  ─────────────────────────────── ← 마루 경계선

  y=72%  [매니저 캐릭터 하단]
         ┌────────┐  ← 품계석 (24×16px, #6B7280)
         └────────┘
  y=75%  ─────────────────────────────── ← 마루 경계선

  품계석은 캐릭터 컴포넌트 (ManagerCharacter.tsx) 내부에서 렌더:
    position: absolute
    bottom: -16px
    left: 50%, transform: translateX(-50%)
    width: 24px, height: 16px
    fill: #6B7280, stroke: #4B5563
```
