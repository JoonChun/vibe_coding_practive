# 코딩 몰라도 AI로 웹사이트 만들기 — 초보자 실전 가이드 (2026)

> 복잡한 코드 대신, AI와 대화만으로 포트폴리오 사이트를 완성했습니다. 코딩 경험이 전혀 없어도 따라할 수 있는 4단계 가이드입니다.

- **난이도**: 초보자 가능
- **소요 시간**: 약 3시간
- **비용**: 무료 (모든 도구 무료 플랜 사용)

---

## 바이브 코딩이란? — AI에게 말로 설명하는 개발 방식

**바이브 코딩(Vibe Coding)**은 AI에게 내가 원하는 것을 자연어로 설명하면, AI가 코드를 자동으로 생성해주는 새로운 개발 방식입니다. 프로그래밍 문법을 외우거나 에러 메시지와 씨름할 필요 없이, **"이렇게 만들어줘"**라는 대화만으로 결과물을 만들 수 있습니다.

기존 노코드(No-Code) 도구와 다른 점은, 바이브 코딩은 **실제 코드가 생성**된다는 것입니다. 템플릿에 갇히지 않고 원하는 기능을 자유롭게 추가할 수 있습니다.

이 글에서는 제가 실제로 바이브 코딩을 활용해 개인 포트폴리오 사이트를 만든 전 과정을 공유합니다. 코딩을 전혀 모르는 분도 이 글을 따라하면 나만의 웹사이트를 만들 수 있습니다.

---

## 사용한 AI 코딩 도구 — 모두 무료

| 도구 | 역할 | 비용 |
|------|------|------|
| [**Gemini**](https://gemini.google.com) | 프로젝트 기획서(PRD) 작성 | 무료 |
| [**Antigravity**](https://antigravity.dev) | AI 코딩 환경 (코드 생성 및 실행) | 무료 플랜 |
| [**Claude**](https://claude.ai) | 코드 작성 및 디버깅 AI | 무료 플랜 |
| [**Vercel**](https://vercel.com) | 웹사이트 무료 배포 및 호스팅 | 무료 |

코딩 없이 웹사이트를 만들기 위해 필요한 것은 이 4가지 도구와 **아이디어**뿐입니다.

---

## 1단계: Gemini로 기획서(PRD) 작성하기

코딩 없이 AI로 웹사이트를 만들 때 가장 중요한 첫 단계는 **"뭘 만들 건지"를 명확하게 정리**하는 것입니다. 이 설계도를 **PRD(Product Requirements Document)**라고 합니다.

저는 Gemini에게 다음과 같이 요청했습니다:

> 인스타그램 프로필에 넣을 링크 페이지를 만들고 싶어.
> 내가 AI로 만든 프로젝트들을 모아서 보여주는 포트폴리오 사이트야.
> 다크 테마에 보라색 네온 느낌으로 만들고 싶어.
> PRD를 작성해줘.

이 한 마디에 Gemini는 다음 항목들을 체계적으로 정리해줬습니다:

- **프로젝트 개요** — 무엇을, 왜, 누구를 위해 만드는지
- **사용자 시나리오** — SNS 유입 → 탐색 → 전환까지의 흐름
- **핵심 기능 목록** — 프로필 헤더, 프로젝트 카드, CTA 버튼, 어드민 패널
- **기술 스택** — HTML, CSS, JavaScript, Tailwind CSS
- **디자인 가이드** — 색상(#8A2BE2), 폰트(Pretendard), 스타일(Glassmorphism)

### PRD가 왜 중요한가?

설계도 없이 집을 지으면 중간에 무너집니다. AI도 마찬가지입니다. 막연하게 "웹사이트 만들어줘"라고 하면 AI도 막연한 결과를 줍니다. 하지만 **구체적인 PRD가 있으면 AI는 정확히 그대로 만들어줍니다.**

> **팁**: PRD를 작성할 때는 "어떤 사람이, 어떤 목적으로, 어떤 기능을 사용하는지"를 구체적으로 적을수록 좋습니다.

---

## 2단계: AI 코딩 도구로 웹사이트 구현하기

PRD가 완성되면 이제 실제로 웹사이트를 만들 차례입니다.

[Antigravity](https://antigravity.dev)를 열고, AI 모델로 Claude를 선택한 뒤, PRD를 그대로 전달했습니다:

> 이 PRD를 기반으로 웹사이트를 만들어줘.
> HTML, CSS, JavaScript로 구현해줘.
> 모바일에서 잘 보이게 만들어줘.

그러자 Claude가 자동으로 코드를 생성하며 다음 기능들을 만들어냈습니다:

### 메인 페이지
- 프로필 이미지 + 보라색 네온 테두리
- 프로젝트 카드 리스트 (데이터 기반 동적 렌더링)
- 인스타그램/GitHub 소셜 링크
- AI 챗봇 (포트폴리오 큐레이터 역할)

### 어드민 패널 — 서버 없이 관리하기
`Ctrl + M`을 누르면 나타나는 숨겨진 관리자 패널입니다. 프로필 수정, 링크 추가/삭제, 드래그앤드롭 순서 변경까지 가능합니다. 별도 서버 없이 브라우저의 LocalStorage로 동작합니다.

### 바이브 코딩 가이드 페이지
"이제 시작하는 바이브 코딩 프로젝트" CTA 버튼을 누르면 나오는 3단계 가이드 페이지입니다. 다른 사람들도 코딩 없이 AI로 웹사이트를 만들 수 있도록 안내합니다.

## 3단계: 대화로 버그 수정하고 기능 다듬기

AI가 처음부터 완벽한 결과를 주지는 않습니다. 하지만 그것이 바이브 코딩의 장점이기도 합니다 — **코드를 이해할 필요 없이, 문제를 말로 설명하면 AI가 알아서 고칩니다.**

실제로 제가 했던 피드백과 AI의 대응:

| 내가 한 말 | AI가 한 일 |
|-----------|-----------|
| "이 버튼 눌러도 아무 반응이 없어" | 링크 연결 버그를 찾아서 수정 |
| "모바일에서 드래그가 안 돼" | 터치 이벤트 지원 코드 추가 |
| "챗봇이 이전 대화를 기억 못해" | 대화 히스토리 기능 구현 |
| "보안 문제 없는지 확인해줘" | XSS 취약점 발견 및 수정 |

핵심은 **"이게 안 돼", "이거 바꿔줘"**라고 자연어로 말하면 AI가 원인을 분석하고 코드를 수정한다는 것입니다. 코딩 지식이 전혀 없어도 이 과정을 반복하면 완성도 높은 결과물을 만들 수 있습니다.

---

## 4단계: Vercel로 무료 배포하기

완성된 웹사이트를 세상에 공개하는 것도 간단합니다. [Vercel](https://vercel.com)을 사용하면 무료로 나만의 URL을 가진 웹사이트를 배포할 수 있습니다.

```bash
npm i -g vercel    # Vercel CLI 설치
vercel login       # 로그인
vercel --prod      # 배포 실행
```

이 세 줄의 명령어면 끝입니다. 별도의 서버 비용 없이 무료로 웹사이트가 공개됩니다.

**완성된 사이트**: [vi-joon-link.vercel.app](https://vi-joon-link.vercel.app)

## 바이브 코딩을 해보고 느낀 점

바이브 코딩을 하면서 가장 크게 느낀 것은, **코딩 없이 웹사이트를 만드는 것이 정말로 가능해졌다**는 것입니다.

예전에는 "웹사이트를 만들려면 HTML/CSS/JS를 공부하고, 프레임워크를 배우고, 서버를 구성하고..."라는 긴 여정이 필요했습니다. 하지만 지금은 **아이디어와 대화 능력**만 있으면 됩니다.

물론 AI가 만든 코드가 항상 완벽하지는 않습니다. 보안 취약점도 있었고, 모바일에서 작동하지 않는 기능도 있었습니다. 하지만 그것도 AI에게 "이거 문제 있는지 확인해줘"라고 하면 스스로 찾아서 고칩니다.

**바이브 코딩의 핵심은 결국 '좋은 질문'을 하는 능력입니다.** 코드를 잘 짜는 것보다, 내가 원하는 것을 명확하게 설명하는 능력이 더 중요한 시대가 왔습니다.

## 자주 묻는 질문 (FAQ)

### 코딩을 전혀 몰라도 바이브 코딩을 할 수 있나요?
네, 가능합니다. 이 글에서 소개한 전 과정에서 제가 직접 코드를 작성한 적은 없습니다. AI에게 원하는 것을 말로 설명하고, 결과를 확인하고, 수정을 요청하는 것이 전부입니다.

### 바이브 코딩에 비용이 드나요?
이 프로젝트에서 사용한 모든 도구(Gemini, Antigravity, Claude, Vercel)는 무료 플랜으로 충분합니다. 추가 비용 없이 웹사이트를 만들고 배포할 수 있습니다.

### 바이브 코딩과 노코드 도구(Wix, Squarespace)의 차이점은?
노코드 도구는 제공되는 템플릿 안에서만 작업할 수 있지만, 바이브 코딩은 실제 코드가 생성되므로 원하는 기능을 자유롭게 추가할 수 있습니다. 커스텀 AI 챗봇, 어드민 패널 같은 기능은 노코드 도구로는 구현하기 어렵습니다.

### 만든 웹사이트를 수정하고 싶으면 어떻게 하나요?
AI에게 다시 수정을 요청하면 됩니다. "이 부분의 색상을 바꿔줘", "새로운 카드를 추가해줘" 같이 자연어로 말하면 AI가 코드를 수정합니다.
