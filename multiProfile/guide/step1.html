툴 설치하기

바이브 코딩에 필요한 3가지 도구를 준비합니다. 모두 무료로 시작할 수 있어요.

🚀 Antigravity란?

Antigravity는 AI를 활용해 코드를 생성하고, 바로 실행하며 결과를 확인할 수 있는 개발 도구입니다. 코딩을 전혀 몰라도, 만들고 싶은 것을 말로 설명하면 AI가 대신 코드를 작성해줍니다.

📸 Antigravity 메인 화면 스크린샷

설치 방법

1. antigravity.dev에 접속합니다.
2. 운영체제에 맞는 설치 파일을 다운로드합니다.
3. 설치 후 실행하면 끝!

📸 다운로드 페이지 스크린샷

🤖 Claude 계정 만들기

Claude는 Anthropic에서 만든 AI 어시스턴트입니다. 코드를 작성하고 수정하는 데 뛰어난 능력을 가지고 있어요. Antigravity에서 Claude를 연결하면 강력한 코딩 파트너를 얻게 됩니다.

1. claude.ai에 접속합니다.
2. 이메일 또는 Google 계정으로 가입합니다.
3. 무료 플랜으로 시작할 수 있습니다.

📸 Claude 가입 화면 스크린샷
💡 팁: Claude Pro 구독을 하면 더 긴 대화와 빠른 응답을 받을 수 있지만, 무료로도 충분히 시작할 수 있어요.

✨ Gemini API 키 발급

Gemini는 Google의 AI 모델입니다. PRD(기획서)를 작성할 때 Gemini의 도움을 받으면 체계적인 기획이 가능합니다.

1. Google AI Studio에 접속합니다.
2. Google 계정으로 로그인합니다.
3. "Get API Key" 버튼을 클릭하여 키를 발급받습니다.
4. 발급받은 키를 안전한 곳에 저장해두세요.

📸 Google AI Studio API 키 발급 화면
💡 팁: API 키는 비밀번호와 같습니다. 절대 다른 사람과 공유하지 마세요!
