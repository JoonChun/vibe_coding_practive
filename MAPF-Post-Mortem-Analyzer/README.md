# 🧩 MAPF Post-Mortem Analyzer

Multi-Agent Path Finding (MAPF) 알고리즘의 실행 결과를 시각화하고 사후 분석(Post-Mortem)을 수행하는 도구입니다.

## 🚀 프로젝트 개요

- **목적**: 다중 에이전트 환경에서 발생하는 충돌, 교착 상태(Deadlock) 및 비효율적인 경로를 분석하여 알고리즘 개선에 기여합니다.
- **주요 특징**:
  - 대규모 테스트 로그 시각화 지원
  - 정면 충돌 및 순환 데드락 케이스 식별
  - 인터랙티브한 분석 인터페이스

## 🛠️ 기술 스택

- **Frontend**: React
- **Backend/Analysis**: Python, ROS2 기반 로그 분석 모듈
