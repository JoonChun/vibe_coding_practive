# 💜 vibe_ws

AI와 감성(Vibe)으로 구축하는 개인 프로젝트 아카이브 저장소입니다. '바이브 코딩'을 통해 만든 결과물들과 그 과정에서의 학습 기록을 통합 관리합니다.

## 📂 프로젝트 목록

### 1. [multiProfile](./multiProfile)

- **컨셉**: AI를 활용한 개인용 멀티 링크 페이지 (성장형 포트폴리오)
- **핵심 기능**: 프로젝트 아카이브 보드, AI 챗봇(Gemini), 로컬 관리자 패널
- **기술 스택**: HTML, CSS, JavaScript, Tailwind CSS

### 2. [MAPF-Post-Mortem-Analyzer](./MAPF-Post-Mortem-Analyzer)

- **컨셉**: 다중 에이전트 경로 탐색(MAPF) 시각화 및 분석 도구
- **기술 스택**: React, Python, ROS2 (기반 분석)
