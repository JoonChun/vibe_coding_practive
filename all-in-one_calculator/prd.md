# [PRD] 올인원 라이프 계산기 (All-in-One Life Calculator)

**프로젝트 명:** 올인원 라이프 계산기
**버전:** 1.0.0
**타겟 층:** 일반인 및 자산 관리에 관심 있는 직장인

---

## 1. 프로젝트 개요

* **목적:** 단위 변환, 금융 계산(복리, 대출), 세금, 디데이 등 실생활에 파편화된 계산 기능을 하나의 로컬 웹 서비스로 통합.
* **핵심 가치:** "번거로운 검색 없이, 개인정보 노출 없이, 모든 계산을 한곳에서."
* **배경:** 금융 및 생활 계산기는 많으나 광고가 많거나 UI가 통일되지 않아 불편함. 이를 해결하기 위해 정교한 계산 로직과 세련된 UI를 결합한 도구 제공.

## 2. 사용자 시나리오 (핵심: 자산 증식 계획)

* 사용자는 매달 일정 금액을 저축하거나 대출을 상환하는 등 자산 계획을 세움.
* **복리 계산기**로 목표 자산 도달 시간을 계산하고, **대출 상환 계산기**로 이자 부담을 시뮬레이션함.
* 두 결과를 **비교 리포트**에 담아 최적의 금융 전략을 선택하고, 중요한 수치는 **상단 위젯**에 고정하여 관리함.

## 3. 핵심 기능 목록

1. **금융 계산:** 복리/적금(년, 월, 일 단위 정밀 지원), 대출 상환(원리금/원금 균등, 만기 일시), 연봉 실수령액.
2. **생활 계산:** 단위 변환(생활 밀착형), 디데이(목표일 관리 및 날짜 차이 계산).
3. **지능형 리포트:** 계산 결과 간 차액 자동 분석 및 비교 데이터 시각화.
4. **개인화 관리:** 최대 5개의 핵심 지표 상단 고정(Pin), 계산 히스토리 로컬 저장.
5. **실시간 피드백:** 입력 즉시 결과가 변하는 No-Button UI 및 수치 카운트업 애니메이션.

## 4. 기술 스택

* **Framework:** Next.js (App Router)
* **Deployment:** Vercel
* **UI/UX:** shadcn/ui, Tailwind CSS (다크모드 기본)
* **State:** Zustand (계산 데이터 및 리포트 상태 관리)
* **Chart:** Recharts (인터랙티브 자산 그래프)

## 5. 화면 구성 (Responsive Layout)

* **Navigation:** 데스크톱(사이드바), 모바일(하단 탭 또는 햄버거 메뉴 및 그리드 레이아웃).
* **Dashboard:** 상단 핀 위젯 영역 + 하단 최근 계산 내역 카드.
* **Calculator Page:** 좌측(입력 폼: 슬라이더/인풋), 우측(실시간 차트 및 결과 바).
* **Report Page:** 선택된 항목의 비교 테이블 및 인사이트 텍스트.

## 6. 상세 기능 명세

* **정밀 복리 로직:** 일(Day) 단위 입력 시 연 이율을 $r/365$로 환산하여 복리 공식($A = P(1+r/n)^{nt}$) 적용.
* **대출 상환 시뮬레이션:** 회차별 원금/이자 변동 내역을 상세 테이블로 제공 및 중도 상환 효과 고려(옵션).
* **비교 엔진:** "A 시나리오가 B보다 총 ₩N원 유리합니다"와 같은 직관적 문구 자동 생성.

## 7. 디자인 가이드

* **Theme:** 차분한 다크모드 (Background: `Slate-950`, Primary: `Emerald-500`).
* **Typography:** Pretendard (본문), Monospace (숫자 데이터).
* **Interaction:** 입력값 변경 시 300ms 디바운스 적용 후 부드러운 차트 업데이트.

## 8. 제약 사항

* **데이터 보안:** 모든 데이터는 브라우저 LocalStorage에 저장 (서버 전송 없음).
* **정확성 고지:** 실제 금융권 계산 방식(휴일, 단수 처리 등)과 미세한 차이가 있을 수 있음을 UI 하단에 명시.
* **호환성:** 최신 에버그린 브라우저(Chrome, Safari, Edge)에 최적화.
